# LyricScreen

A powerful and fast lyrics management and display app.

## Setup

* [NodeJS][node]
* [CoffeeScript][coffeescript]
* [Bower][bower]
* [Gulp][gulp]

```bash
npm install -g coffee-script gulp bower
npm install
bower install
```

## Running

```bash
gulp watch-serve
```

[coffeescript]: http://coffeescript.org
